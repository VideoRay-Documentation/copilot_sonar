CoPilot Sonar hardware requirements include a VideoRay Pro 4 ROV system and BlueView P- or V- series 900-130 sonar (a 900-90 sonar can be used but may result in reduced performance of the tracking). A stabilizing fin is included with CoPilot sonar to improve tracking performance.

CoPilot Sonar includes software delivered on a flash drive and a HASP hardware license key. If you purchase CoPilot Sonar as part of a Pro 4 system, the software will be installed on the computer. If you purchase CoPilot Sonar separately, you will need to install the software.
