Sonar Display

The sonar display includes the following elements:

  • Sonar image
    • Range grid with range indicators
    • Active Target Tracking (Green Box)
  • Position information
    • ROV
      • Depth
      • Heading
    • Cursor
      • Range
      • Relative Bearing
    • Target
      • Range
      • Relative Bearing

Additional related settings for the sonar and tracking include:

  • Sonar Target Type Toggle
  • Sonar Color Palette
  • Sonar End Range
  • Sonar Target Size
