Software Guide

To start CoPilot Sonar, click on the SeeByte button on the VideoRay Cockpit Control Bar. This will open a virtual machine window. Click on the CoPilot Sonar icon.

Once CoPilot Sonar starts, the operator is presented with a window that provides sonar and depth displays and controls for enabling and disabling tacking, and various other settings and status indicators. Features and use of CoPilot Sonar are described in the following sections.

To stop Copilot Sonar, click on the X in the upper left hand corner of the window. Close the virtual machine window.
