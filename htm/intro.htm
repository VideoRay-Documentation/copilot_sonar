1 - Introduction to VideoRay Sonar CoPilot

This chapter of the user manual provides a general product overview and explains how to install and operate the VideoRay Sonar CoPilot unit and its pilot interface.

1.1 Scope and Purpose

This manual provides the following:

  • an overview of the VideoRay Sonar CoPilot hardware unit configuration and connectivity;
  • a general description and specification of VideoRay Sonar CoPilot pilot interface; and
  • operating instructions for VideoRay Sonar CoPilot and equipment.

1.2 Targeted Audience

This manual is designed and developed for operators of the VideoRay Pro 4 MicroROV with BlueView Sonar who are required to operate the VideoRay remotely operated vehicle (ROV) systems using VideoRay Sonar CoPilot.
